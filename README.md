# replicas-agent

A skill for background coding agents running inside Replicas cloud workspaces.

## Overview

This skill provides guides and references for capabilities available in Replicas workspaces, including:

- **Previews** — Expose locally running services as public preview URLs
- **Slack** — Send messages, read threads, and interact via the Slack Web API
- **Linear** — Fetch issues, update state, and comment via the Linear GraphQL API
- **GitHub** — Use the pre-authenticated `gh` CLI for PRs, issues, and actions
- **Docker** — Start and use the Docker daemon for containerized workflows
